O arquivo extractors_and_classifiers.py contém as funções referentes aos extratores de atributos LBP, Momentos de HU
e GLCM. Além das funções dos classificadores de padrões GaussianNB e SVM.

Já o arquivo cnn_digits_recognition.py contém a implementação de uma rede neural convolucional para o reconhecimento
dos dígitos da base disponibilizada para o trabalho.
